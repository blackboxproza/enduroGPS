❓ Connection Help

📱 For Mobile Devices:

  • Scan the QR code on your GPS device screen
  • Or manually connect to the WiFi network shown on device
  • Your browser should automatically open this page

💻 For Computers:

  • Connect to the WiFi network shown on your GPS device
  • Open a web browser and navigate to the IP address shown
  • This page will load automatically

🔧 Troubleshooting:

  • Ensure you're connected to the GPS device's WiFi network
  • Try refreshing this page if files don't load
  • For large files, be patient during upload/download
  • Close other apps that might interfere with the connection
